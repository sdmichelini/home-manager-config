# Custom Shortcuts

## Tmux

Prefix key is `Ctrl+a` (not the default `Ctrl+b`).

| Shortcut | Action |
|----------|--------|
| `Ctrl+a` | Prefix key |
| `prefix r` | Reload tmux config |
| `prefix \|` | Split pane horizontally |
| `prefix -` | Split pane vertically |

## Neovim

Leader key is `\` (backslash).

| Shortcut | Action |
|----------|--------|
| `\ff` | Find files (telescope) |
| `\fg` | Live grep (telescope) |
| `\fh` | Search help tags (telescope) |
| `Ctrl+p` | Find git files (telescope) |
