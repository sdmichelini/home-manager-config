# Custom Shortcuts

## Tmux

Prefix key is `Ctrl+a` (not the default `Ctrl+b`).

| Shortcut | Action |
|----------|--------|
| `Ctrl+a` | Prefix key |
| `prefix r` | Reload tmux config |
| `prefix \|` | Split pane horizontally |
| `prefix -` | Split pane vertically |

## Neovim

Leader key is `\` (backslash).

| Shortcut | Action |
|----------|--------|
| `\ff` | Find files (telescope) |
| `\fg` | Live grep (telescope) |
| `\fh` | Search help tags (telescope) |
| `Ctrl+p` | Find git files (telescope) |
| `\a` | Add file to harpoon list |
| `Ctrl+e` | Toggle harpoon quick menu |
| `Ctrl+h` | Select harpoon file 1 |
| `Ctrl+t` | Select harpoon file 2 |
| `Ctrl+n` | Select harpoon file 3 |
| `Ctrl+s` | Select harpoon file 4 |
| `Ctrl+Shift+p` | Previous harpoon buffer |
| `Ctrl+Shift+n` | Next harpoon buffer |
